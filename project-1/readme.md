# 🚀 Advanced Todo List in Java (Core Java + DSA)

A feature-rich command-line Todo List application built using **Core Java**, **OOP**, and **Data Structures & Algorithms**.

## ✅ Features

### 📌 CRUD Operations
- [X] View Tasks
- [X] Add Task
- [ ] Remove Task
- [ ] Update Task

### 🔍 Searching & Filtering
- [ ] Search Task
- [ ] View Completed Tasks
- [ ] View Pending Tasks

### ✔️ Status Operations
- [ ] Mark Task as Completed
- [ ] Mark Task as Pending

### 📅 Sorting
- [ ] Sort by Deadline
- [ ] Sort Alphabetically

### 📊 Statistics
- [ ] Show Total Tasks
- [ ] Show Completed Count
- [ ] Show Pending Count

### ↩️ Undo / Redo
- [ ] Undo Last Operation
- [ ] Redo Last Operation

### 💾 File Operations
- [ ] Save Tasks
- [ ] Load Tasks
- [ ] Export Tasks

### ⭐ Advanced Features
- [ ] Delete All Tasks
- [ ] Clear Completed Tasks
- [ ] Set Task Priority
- [ ] Change Deadline
- [ ] View Overdue Tasks

---

## 🛠️ Tech Stack

- Core Java
- Object-Oriented Programming (OOP)
- Collections Framework (`ArrayList`)
- Java Time API (`LocalDate`)
- Exception Handling
- File Handling
- Data Structures & Algorithms (DSA)
