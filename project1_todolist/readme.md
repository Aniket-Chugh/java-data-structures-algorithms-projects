# 🚀 Advanced Todo List in Java (Core Java + DSA)

A feature-rich command-line Todo List application built using **Core Java**, **OOP**, and **Data Structures & Algorithms**.

## ✅ Features

### 📌 CRUD Operations
- [X] View Tasks
- [X] Add Task
- [X] Remove Task
- [X] Update Task

### 🔍 Searching & Filtering
- [X] Search Task
- [X] View Completed Tasks
- [X] View Pending Tasks

### ✔️ Status Operations
- [X] Mark Task as Completed


### 📅 Sorting
- [X] Sort by Deadline
- [X] Sort Alphabetically

### ↩️ Undo / Redo
- [ ] Undo Last Operation

### ⭐ Advanced Features
- [X] Delete All Tasks
- [ ] Clear Completed Tasks


---

## 🛠️ Tech Stack

- Core Java
- Object-Oriented Programming (OOP)
- Collections Framework (`ArrayList`)
- Java Time API (`LocalDate`)
- Exception Handling
- File Handling
- Data Structures & Algorithms (DSA)
