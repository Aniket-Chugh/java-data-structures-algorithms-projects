# 📞 PhoneBook DSA Project (Java)

A feature-rich **terminal-based PhoneBook application** built entirely in **Core Java** to master **Data Structures & Algorithms** through real-world implementation instead of solving isolated coding problems.

This project focuses on implementing common DSA concepts from scratch while building a practical application similar to a real phonebook.

---

# 🎯 Objective

The goal of this project is to strengthen Java and DSA fundamentals by applying them in a real application.

Instead of building a simple CRUD application, this project demonstrates how different data structures and algorithms work together to solve real problems efficiently.

# 🧠 Data Structures & Algorithms Used

| Concept                | Purpose                                         |
| ---------------------- | ----------------------------------------------- |
| Arrays / Dynamic Array | Store contacts efficiently                      |
| Merge Sort             | Sort contacts alphabetically or by phone number |
| Binary Search          | Fast searching on sorted contacts               |
| Trie                   | Prefix search & auto-complete suggestions       |
| Stack                  | Undo / Redo operations                          |
| Queue                  | Maintain recent searches                        |
| Max Heap               | Track most contacted users                      |

# 📈 Time Complexity

| Operation      | Complexity      |
| -------------- | --------------- |
| Add Contact    | O(1)*           |
| Delete Contact | O(n)            |
| Update Contact | O(log n) / O(n) |
| Binary Search  | O(log n)        |
| Merge Sort     | O(n log n)      |
| Trie Insert    | O(L)            |
| Trie Search    | O(L)            |
| Undo           | O(1)            |
| Redo           | O(1)            |
| Recent Search  | O(1)            |
| Most Contacted | O(log n)        |

> *Amortized complexity when using a dynamic array.

# 🎯 Why This Project?

Most DSA practice involves solving disconnected coding problems.

This project takes a different approach by integrating multiple data structures and algorithms into a single, practical application. It demonstrates how concepts like sorting, searching, stacks, queues, heaps, and tries can work together in a real-world system, making it an excellent learning resource for Java developers and interview preparation.
